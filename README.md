## Описание проекта

В последние десятилетия гималайский альпинизм превратился в массовое и коммерциализированное явление. При этом восхождения остаются деятельностью с повышенным уровнем риска.

Данный проект представляет собой комплексное статистическое исследование факторов, влияющих на **успешность** и **безопасность** гималайских альпинистских экспедиций, с применением методов **Big Data** и **статистической аналитики**.

Проект переводит сырые данные из Himalayan Database в структурированные аналитические выводы с помощью современных Python-библиотек.

### Бизнес-ценность

- 📊 Выявление ключевых факторов успеха экспедиций
- 🎯 Оценка рисков для планирования восхождений
- 📈 Количественная оценка влияния коммерческих маршрутов
- 🔬 Статистически обоснованные рекомендации для альпинистов

## Цель исследования

**Основная цель** — анализ факторов, влияющих на успешность и безопасность гималайских альпинистских экспедиций, с применением методов описательной статистики, регрессионного анализа и критериев согласия.

### Задачи

1. Описать структуру исходных данных и выполнить предобработку
2. Сформулировать и реализовать методику проверки статистических гипотез
3. Провести вычислительный эксперимент и интерпретировать результаты

## Гипотезы

В работе проверяется **7 статистических гипотез**:

| № | Гипотеза | Описание |
|:---:|:---|:---|
| **H₁** | **Влияние высоты** | С увеличением высоты вершины вероятность успешного восхождения снижается |
| **H₂** | **Сезонность** | Доля успешных экспедиций зависит от сезона |
| **H₃** | **Использование кислорода** | Экспедиции, использующие кислород, имеют более высокий уровень успешности |
| **H₄** | **Продолжительность** | Средняя длительность экспедиций различается по причинам завершения и типам маршрутов |
| **H₅** | **Наёмный персонал** | Экспедиции с наёмным персоналом имеют более низкий уровень смертности |
| **H₆** | **Коммерческие маршруты** | На коммерческих и стандартных маршрутах вероятность успеха существенно выше |
| **H₇** | **Популярность вершин** | Количество экспедиций на вершину зависит от её высоты |

## 📊 Данные

### Источник данных

Проект использует **агрегированный датасет** гималайских экспедиций, подготовленный в рамках проекта **TidyTuesday** на основе **Himalayan Database**.

### Структура данных

| Датасет | Описание | Количество записей |
|:---|:---|:---:|
| `exped_tidy.csv` | Сведения об экспедициях | 882 |
| `peaks.csv` | Информация о вершинах | 116 |

### Ключевые поля

| Таблица | Поля |
|:---|:---|
| **Экспедиции** | `EXPID`, `PEAKID`, `YEAR`, `SEASON_FACTOR`, `TOTDAYS`, `TERMREASON_FACTOR`, `O2USED`, `MDEATHS`, `HDEATHS`, `COMRTE`, `STDRTE`, `NOHIRED` |
| **Вершины** | `PEAKID`, `PEAKNAME`, `HEIGHTM`, `REGION` |

### Производные признаки

| Признак | Описание |
|:---|:---|
| `SUCCESS` | Бинарный индикатор успеха экспедиции |
| `DEATHS` | Суммарное число погибших (участники + наёмный персонал) |
| `ANY_DEATH` | Индикатор наличия смертельных случаев |
| `HIRED` | Индикатор наличия наёмного персонала |
| `COM_OR_STD` | Индикатор коммерческого/стандартного маршрута |
| `N_EXP` | Количество экспедиций на вершину |
| `LOG_N_EXP` | Логарифм числа экспедиций |

## Методология и результаты

### Результаты проверки гипотез

| № | Гипотеза | Критерий | Ключевые результаты | Значимость | Вывод |
|:---:|:---|:---|:---|:---:|:---|
| **H₁** | Влияние высоты на успех | Логистическая регрессия | Коэффициент: **отрицательный**, p < 0.05 | ✅ **Значимо** | Высота негативно влияет на успех |
| **H₂** | Различия по сезонам | Хи-квадрат (χ²) | χ² = 17.15, df = 3, p = 0.00066 | ✅ **Значимо** | Сезон значимо влияет на успех |
| **H₃** | Влияние кислорода | Хи-квадрат + логистическая регрессия | χ² = 160.82, p < 0.001 <br> OR = **12.84** | ✅ **Значимо** | Кислород повышает успех в 12.8 раз |
| **H₄** | Длительность по причинам | t-тест (Уэлча) | t = 0.59, p = 0.56 | ❌ **Не значимо** | Различий в длительности нет |
| **H₅** | Наёмный персонал и смертность | Хи-квадрат + логистическая регрессия | χ² = 1.55, p = 0.21 <br> OR = 2.38, p = 0.15 | ❌ **Не значимо** | Влияние на смертность не обнаружено |
| **H₆** | Коммерческие маршруты | Хи-квадрат + множественная логистическая регрессия | χ² = 126.36, p < 0.001 <br> OR = **10.50** | ✅ **Значимо** | Коммерческие маршруты в 10+ раз повышают успех |
| **H₇** | Популярность вершин | Корреляция + линейная регрессия | r_Пирсона = 0.47, r_Спирмена = 0.36 <br> R² = 0.38, p < 0.001 | ✅ **Значимо** | Высокие вершины посещаются чаще |
